
# Nima Omme Special (Python/Flask-version)

## Så här laddar du upp på Render:

1. Gå till https://render.com
2. Skapa konto och klicka på 'New' → 'Web Service'
3. Koppla till ditt GitHub-konto och välj detta projekt (eller ladda upp som nytt repo)
4. Render läser automatiskt in 'render.yaml'
5. Klart! Du får en länk som du kan dela

## Lokalt (testa på egen dator):

1. Installera Python 3
2. Öppna terminal och kör:
   pip install flask
   python server.py
3. Gå till http://localhost:5000 i din webbläsare
